# AI Resume Screening System

 Live Demo
https://ai-resume-screening-system-jsqzivcrhhsejpizqv9qke.streamlit.app/

 Overview
The AI Resume Screening System is a web application that analyzes resumes by extracting text from uploaded resume images and comparing candidate skills with a job description using semantic similarity. The application identifies matched skills, missing skills, and calculates an overall compatibility score to assist in the resume screening process.

 Features

 Upload resume images (JPG, JPEG, PNG)
 Extract text using EasyOCR
 Automatic skill extraction
 Semantic skill matching using Sentence Transformers (MiniLM)
 Resume-job match score calculation
 Missing skill detection
 Interactive Streamlit interface

Tech Stack
 Python
 Streamlit
 EasyOCR
 Sentence Transformers
 Pillow

## Run Locally

Clone the repository

```bash
git clone https://github.com/Reham6133/AI-Resume-Screening-System.git
```

Move into the project directory

```bash
cd AI-Resume-Screening-System
```

Install the required packages

```bash
pip install -r requirements.txt
```

Run the application

```bash
 run app.py
```
